# Quizter

Quizter is a fun quiz app for mobile.


## Setup
Before start set up secrets.
- lib/auth/<example_top_secret.dart>. 
- Make a copy in the same place. 
- lib/auth/<top_secret.dart>. 
- There is a demo key for testing in the example file.
